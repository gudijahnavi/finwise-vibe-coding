# User Manual — SHOPEZ

## For Customers

### 1. Creating an Account
Click **Sign Up** → enter name, email, password → submit. You'll be redirected to login.

### 2. Browsing Products
Use the search bar or category filters on the homepage. Click any product card to
view full details, price, stock, and reviews.

### 3. Adding to Cart
On a product page, choose quantity and click **Add to Cart**. View your cart via the
cart icon in the top navigation.

### 4. Checkout
From the cart, click **Checkout** → select or add a delivery address → choose a
payment method → confirm payment. You'll see an order confirmation on success.

### 5. Tracking an Order
Go to **My Orders** → select an order to see its current status:
`Placed → Packed → Shipped → Delivered`.

### 6. Wishlist & Reviews
Click the heart icon on any product to save it to your wishlist. After delivery, you
can leave a rating and review from the order detail page.

## For Admins

### 1. Accessing the Admin Dashboard
Log in with an admin account → navigate to `/admin`.

### 2. Managing Products
**Products** tab → Add/Edit/Delete products, update stock and pricing.

### 3. Managing Orders
**Orders** tab → view all customer orders → update status as items are packed/shipped/delivered.

### 4. Managing Users
**Users** tab → view registered customers (read-only or role management, per your build).

## Troubleshooting
| Issue | Fix |
|---|---|
| Can't log in | Check email/password; use "Forgot Password" |
| Payment fails | Retry with a valid sandbox test card; check network |
| Item shows out of stock | Admin needs to restock via the Products tab |
