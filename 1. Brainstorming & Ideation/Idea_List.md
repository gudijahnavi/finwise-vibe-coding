## Selection Rationale
**SHOPEZ** was selected because it:
- Covers the broadest set of full-stack skills required by the internship track (auth, CRUD, payments, state management, DB relations, admin dashboard, deployment).
- Has high feasibility within the internship timeframe using a standard MERN-style stack.
- Maps directly to a well-understood, evaluable customer problem (see Problem Statement).
- Is easily demoable end-to-end (browse → cart → checkout → order tracking → admin panel).
