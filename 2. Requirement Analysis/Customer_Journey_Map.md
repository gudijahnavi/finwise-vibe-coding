# Customer Journey Map — SHOPEZ

| Stage | User Action | Touchpoint | Thoughts | Feelings |
|---|---|---|---|---|
| **1. Discover** | Lands on SHOPEZ via search/ad/link | Homepage, product listing | "Let's see what they have" | Curious |
| **2. Browse** | Searches/filters products by category, price, rating | Search bar, filter panel, product cards | "Can I find exactly what I want quickly?" | Neutral → Engaged |
| **3. Evaluate** | Opens product detail page, checks images, reviews, price | Product Detail Page (PDP) | "Is this good value? Is it trustworthy?" | Cautiously interested |
| **4. Add to Cart** | Adds item(s), adjusts quantity | Cart icon, cart drawer/page | "Let me see the total before deciding" | Building confidence |
| **5. Authenticate** | Logs in / signs up / continues as guest | Login/Signup modal | "I hope this is quick" | Slight friction |
| **6. Checkout** | Enters/selects address, picks payment method, places order | Checkout page, payment gateway | "Is this secure?" | Anxious → Relieved on success |
| **7. Confirmation** | Sees order confirmation, gets email/notification | Order success page | "Great, it's done" | Satisfied |
| **8. Track** | Revisits site to check order status | Order history / tracking page | "Where is my order now?" | Anticipation |
| **9. Receive & Review** | Gets product, optionally leaves a review | Delivery, review form | "Did it meet expectations?" | Satisfied / Disappointed |
| **10. Return (repeat)** | Comes back for next purchase | Homepage, wishlist, recommendations | "Was that easy enough to do again?" | Loyal (if positive) |

## Key Insight
The two highest-drop-off stages are **Checkout (friction)** and **Track (lack of visibility)**
— SHOPEZ's core feature set (streamlined checkout + live order status) is designed
directly around fixing these two moments.
